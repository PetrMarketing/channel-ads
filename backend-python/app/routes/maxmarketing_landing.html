Вопросы

Часто спрашивают

01Нужно ли платить за подключение?+
Нет. Регистрация бесплатна. Первые 2 дня для нового канала — полностью бесплатно. Дальше — единый тариф 490 ₽/месяц.
02Какие платёжные системы поддерживаются?+
Tinkoff, YooKassa, Prodamus, Robokassa. Подключаются за 5 минут через ваш аккаунт у платёжки. Деньги клиентов идут напрямую к вам.
03Как подключить канал MAX?+
Найдите бота PKMarketing в MAX → /start → откройте личный кабинет → добавьте бота PKMarketing администратором вашего канала. Канал появится автоматически.
04Можно ли использовать на нескольких каналах?+
Да. Тариф 490 ₽/месяц считается за каждый подключённый канал. Можно подключить сколько угодно каналов — у каждого своя статистика, настройки и воронки. Первые 2 дня для каждого нового канала бесплатно.
05Что такое ИИ-токены?+
Виртуальная валюта для ИИ-функций: лендинги, аватары, описания. Часть токенов уже включена в тариф, докупить можно отдельно.
