# Скриншоты для первой партии статей (5 ВЧ-статей)

Всего нужно **29 уникальных скриншотов**. Каждый загружается в админку
**один раз** через **Админка → Блог → Скриншоты → + Добавить**, slug
указывается ровно как ниже — после загрузки скриншот автоматически
появится во всех статьях, где он используется.

Домен в скриншотах должен быть **`max.pkmarketing.ru`** (не itcakes.ru).
Бот в скриншотах — **ПКРеклама**.

## Workflow

1. Зайти **Админка → Блог → Скриншоты** на `max.pkmarketing.ru/admin/blog`
2. Нажать **+ Добавить скриншот**
3. Загрузить файл, заполнить:
   - **Slug** ← скопировать из таблицы выше (например `max-create-channel-button`)
   - **Название** ← краткое из колонки «Что на скриншоте»
   - **Alt-текст** ← фраза для SEO (можно расширенно)
4. Сохранить — он автоматически подтянется в нужные статьи
5. После загрузки всех скриншотов открыть **Блог → Статьи**, проверить что в колонке «Скрин-ты» вместо ⚠️ стоит число → опубликовать
